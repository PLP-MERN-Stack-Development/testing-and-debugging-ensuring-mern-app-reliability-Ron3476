# MERN Testing & Debugging Template

This repository contains a complete testing and debugging starter kit for a MERN stack app.
It includes:
- Jest unit & integration examples (server)
- Supertest + mongodb-memory-server integration test
- React Testing Library unit test (client)
- Cypress end-to-end example
- Error boundary, global error handler, and logger examples
- CI workflow (GitHub Actions)
- Documentation of testing strategy

Structure (monorepo):
- client/
- server/
- cypress/
- docs/
- .github/workflows/

See `docs/testing-strategy.md` for guidance.
